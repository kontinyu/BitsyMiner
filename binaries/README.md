## Binary installation files

You can use a web-based flasher to program your device.

I suggest either [this one](https://esptool.spacehuhn.com/) or [this one](https://espressif.github.io/esptool-js/).

<br><br>
First, save these files to your drive:

<ul>
  <li>boot_app0.bin</li>  
  <li>bootloader.bin</li>
  <li>partitions.bin</li>
</ul>




<br><br>
Next, save the file that matches your device to your drive. This will be one of the following:

<ul>
  <li>2.4inch_9341.bin</li>
  <li>2.8inch_9341.bin</li>
  <li>2.8inch_ST7789.bin</li>
  <li>headless.bin</li>
</ul>


<br><br>
Navigate to the web flasher of your choice and set the each file to the locations shown below and flash away:

![Image of web flasher](/assets/esptool_image.png)
